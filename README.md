# Taja Fal Sabji House

An AI-powered full-stack grocery delivery platform inspired by Blinkit and Zomato.

## Features

- Google Login
- Mobile OTP Login
- Admin Dashboard
- Live Order Tracking
- AI Product Recommendations
- Dynamic Pricing
- Smart Cart Suggestions
- Invoice Generation
- Responsive UI

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Node.js
- Express.js
- MongoDB Atlas
- Firebase Authentication
- Cloudinary
- FastAPI
- Python
- scikit-learn
